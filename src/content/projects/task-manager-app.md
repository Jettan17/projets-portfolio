## About This Project

A comprehensive task management solution designed for both individual productivity and team collaboration. Features real-time synchronization and an intuitive drag-and-drop interface.

## Key Features

- **Real-time Updates**: Changes sync instantly across all connected clients using Socket.io
- **Drag and Drop**: Intuitive task organization with smooth animations
- **Team Collaboration**: Share boards, assign tasks, and track progress together
- **Due Dates & Reminders**: Never miss a deadline with smart notifications
- **Labels & Filters**: Organize tasks with custom labels and powerful filtering

## Technical Highlights

Built with a React frontend and Node.js/Express backend, this application demonstrates proficiency in:

- State management with Redux Toolkit
- RESTful API design with comprehensive error handling
- MongoDB aggregation pipelines for efficient data queries
- WebSocket implementation for real-time features
- JWT authentication with refresh token rotation
