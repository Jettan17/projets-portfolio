## About This Project

A weather application that combines accurate forecasting data with stunning visualizations. Users can check weather conditions for any location worldwide with detailed hourly and weekly forecasts.

## Key Features

- **Location Detection**: Automatic weather for your current location using the Geolocation API
- **Global Search**: Find weather for any city in the world
- **Interactive Charts**: Visualize temperature, precipitation, and wind data
- **7-Day Forecast**: Plan ahead with extended weather predictions
- **Weather Maps**: Interactive radar and satellite imagery

## Technical Highlights

This project showcases integration with multiple external APIs:

- OpenWeatherMap API for weather data
- Mapbox for interactive map displays
- Chart.js for data visualization
- Browser Geolocation API for location services

The application handles API rate limiting, caching strategies, and graceful error handling for network failures.
